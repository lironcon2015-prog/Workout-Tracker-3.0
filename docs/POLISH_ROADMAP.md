## P1 — טיפוגרפיה

**טוקנים (להוסיף ל-`:root`):**
```css
--fs-xs: .65rem;   /* מטא, תגיות, יחידות */
--fs-sm: .72rem;   /* טקסט משני, תתי-כותרות */
--fs-md: .8rem;    /* גוף */
--fs-lg: .9rem;    /* גוף מודגש, שורות רשימה ראשיות */
--fs-xl: 1.05rem;  /* כותרות סקשן */
--fs-2xl: 1.3rem;  /* כותרות מסך */
--fs-hero: 1.9rem; /* מספרי hero (בית, טבעת) — טיימר-מגה נשאר חופשי */
```

**מיפוי (borderline מוכרע ויזואלית per-component):**
| היום | יעד |
|------|-----|
| ≤0.68rem | --fs-xs |
| 0.7–0.78rem | --fs-sm |
| 0.8–0.85rem | --fs-md |
| 0.9–0.95rem | --fs-lg |
| 1–1.15rem | --fs-xl |
| 1.2–1.4rem | --fs-2xl |
| כל `em` | המרה ידנית ל-rem לפי הקשר (לחשב את הערך האפקטיבי בפועל!) |

**משקלים:** 900 = כותרות-ענק + מספרי hero · 700 = כותרות/הדגשה חזקה · 600 = הדגשת שורה ·
400 = גוף. **800 מתחסל** (→900 למספרים, →700 לטקסט). 200/300/500 → 400 או 600 לפי הקשר.

**tabular-nums:** utility `.num { font-variant-numeric: tabular-nums; }` + החלה ישירה על:
טיימרים (session strip, מנוחה, Live), פיקרים (משקל/חזרות/RIR), KPI (בית, Composition, אנליטיקה),
kcal ביומן (טבעת, שורות, preview), רשימת שקילות, ציר Y בגרפים.

**סיכון מרכזי:** איחוד 0.9→--fs-lg עלול לשבור שורות צפופות (session-log, טבלת TDEE). בדיקה פרטנית שם.

## P3 — Radius + ריווח

**Radius — סולם סופי:** `--r-xs: 8px` (חדש — צ'יפים/תגיות) · `--r-sm: 14px` · `--r-md: 20px` ·
`--r-lg: 28px` · pill 9999px · עיגול 50%.

| היום | יעד |
|------|-----|
| 6, 8px | --r-xs |
| 10, 12, 14px | --r-sm |
| 16px, 1rem, 18, 20px | --r-md |
| 2rem, 28px | --r-lg |
| 50px, 9999px | pill (9999px) |
| 50% | נשאר (עיגולים) |

**כלל קינון:** radius פנימי ≤ חיצוני − הפרש padding (ויזואלית: פנימי קטן מחיצוני).

**ריווח:** רשת מותרת {4, 8, 12, 16, 20, 24} + {2} למיקרו (אייקון↔טקסט צמוד).
מיפוי: 3→4 · 5→4 · 6→8 (זהירות: מרווח אייקונים צפוף — לשפוט ויזואלית, 6→4 אם 8 מנפח) ·
7→8 · 9→8 · 10→8 או 12 · 13,14→12 · padding באותה רשת.

## P4 — מושן

**טוקנים:**
```css
--dur-1: 120ms;  /* לחיצה, hover, טוגלים */
--dur-2: 200ms;  /* fades, החלפות תוכן */
--dur-3: 320ms;  /* sheets, מעברי מסך */
--ease: cubic-bezier(.2, 0, 0, 1);        /* standard decelerate */
--ease-spring: cubic-bezier(.34, 1.3, .4, 1); /* sheets בלבד */
```

- כל 130 ה-transitions → אחד משלושת המשכים + `--ease`.
- **מצב לחיצה אחיד:** כל אלמנט אינטראקטיבי (כפתורים, שורות רשימה, כרטיסים לחיצים, צ'יפים):
  `:active { transform: scale(.96-.98); }` עם `--dur-1`. לוודא שלא מתנגש ב-transform קיים
  (למשל sheets עם translateY — שם opacity בלבד).
- keyframes קיימים (18) נשארים; רק ליישר משכים בולטים לסולם.
- `prefers-reduced-motion: reduce` — קיים חלקית; לוודא כיסוי גלובלי ל-transitions החדשים.

**סיכון:** scale על אלמנט עם overflow/scroll עלול ליצור ריצוד ב-iOS — לבדוק על שורות רשימה ארוכות.

## P5 — ליטושי פרימיום

1. **רשימות hairline:** שורות יומן מזון, רשימת שקילות, היסטוריית תזונה, ארכיון — קונטיינר אחד
   עם `border` + מפרידי `border-bottom: 1px solid var(--hairline)` פנימיים, במקום קופסה+רקע לכל שורה.
   (ראה מוקאפ.) **זהירות:** onclick על שורות נשאר זהה — שינוי CSS בלבד.
2. **גרבר אחיד ל-bottom sheets:** ידית 36×4px `--surface-4` בראש כל sheet (יש בחלק — לוודא בכולם).
3. **Empty states:** `emptyStateHtml()` הגלובלי קיים — לוודא שכל מסך ריק משתמש בו (יומן, ארכיון,
   אנליטיקה, שקילות, חיפוש מזון) עם אייקון+כותרת+פעולה באותה תבנית.
4. **Safe-area audit:** padding-bottom של תוכן גליל מעל tab-bar/session-strip בכל המסכים
   (כולל overlay יומן המזון) — `env(safe-area-inset-bottom)` עקבי.
5. **אופטיקה של אייקונים:** `Material Symbols` — לוודא `font-size` מסולם קבוע (1rem/1.2rem/1.5rem)
   ויישור אנכי אחיד (flex align-center) בכל שורות האייקון+טקסט.
